* Oracle *
Zainstaluj Oracle DataBase 21c Express Edition.
Do importu bazy danych możesz skorzystać z SQL*Plus bądz z apliakcji sqldeveloper.
Po imporcie zobaczysz tabele w bazie: FAKTURA, FIRMA, KLIENT, POZYCJA_FAKTURY, STAWKA_VAT, USLUGA, UZYTKOWNIK.
